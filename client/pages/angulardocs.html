Directives

These are the fun little extension to Html that Angular gives us. These start with ng and are attach a special behavior to the elements. We use this on the home page to call are random function and to bind the variables that are the facts to the element.

Controller

Where the logic goes in the app. This is going to be your powerhouse and where your app will get functionallity. We use this in the homepage to make a simple Math.random() function to decide which fact show.

Services

These are to modularize your code and keep the controllers clean. We store functions and data we may use over again in Services. In this app we aren't using any custom Services but use some of Angulars built in handy Services. We use the $locationProvider to set the mode to Html5 and $urlProvider to / so that the # is removed in our url.

Scope

This is simply the realm of relevancy. This is where information has value and can be manipulated to form our app.

Expression

How data in the scope is displayed as the html elements. We use this as basically a dynamic placeholder in this app to grab the different facts to display.

Veiw

Simply what we are seeing in this app. This is just the user interface.

Data Binding

Angular utilizes two way data binding meaning we can take information from the model and veiw and pass them back and forth. Here you can see it in this addition text box. You can inter the values and it will calculate it as you go.
